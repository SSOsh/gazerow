# 메뉴바 상태 아이콘 에셋 통일

> 작성: suho.do · 2026-07-17 · 상태: 완료

## 목표

- 메뉴바의 gazerow 아이콘을 앱 에셋과 같은 3×3 키캡 + 시선 링 모티프로 변경한다.
- 18px 단색 template 아이콘에서도 키보드와 시선 선택 의미가 식별되도록 단순화한다.
- macOS 메뉴바의 밝은/어두운 배경에 자동 적응하는 template 속성을 유지한다.

## 작업 목록

### Phase 1: 상태 아이콘

- [x] 1.1 기존 키캡 + 재생 커서 도형을 3×3 키보드 격자로 변경
- [x] 1.2 우상단 키를 선택하는 시선 링과 중심점 추가

### Phase 2: 검증

- [x] 2.1 픽셀 렌더링 기반 회귀 테스트 추가
- [x] 2.2 전체 테스트와 실제 앱 재실행 확인

## 완료 조건

- 메뉴바 아이콘이 AppIcon의 핵심 모티프와 일치한다.
- 18px template 이미지와 접근성 설명이 유지된다.
- 관련 테스트와 전체 테스트가 통과한다.
